# jamescochran.github.io

Personal GitHub Pages site — a minimal landing page with links to projects and socials.

## Contents

### `/` — Landing page
A single-page link hub (`index.html`) with avatar, name, and links to projects, Bluesky, and Ko-fi.

### `/TwinStarsRando` — Twin Stars Randomizer
A free, unofficial fan-made companion app for the card game **Twin Stars Adventure Series I** (Button Shy Games). Single-file PWA — download and open in any browser, no installation or account required.

Features:
- Random or manual character/scenario selection
- In-session timer
- Persistent play history and results tracking

### `/Recipes` — Recipes
Static recipe pages.

## Tech

Plain HTML/CSS/JS. No build step, no dependencies. Deployed automatically via GitHub Pages.
